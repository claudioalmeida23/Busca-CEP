# 📍 Busca CEP com Mapa

Aplicação web simples que permite **buscar um CEP e visualizar sua localização no mapa**, além de abrir a navegação diretamente no GPS do Google Maps.

O projeto utiliza **Leaflet com OpenStreetMap**, consulta dados de CEP pela **BrasilAPI** e realiza geocodificação usando **Nominatim**.

---

## 🚀 Funcionalidades

* 🔎 Busca de endereço através do **CEP**
* 🗺️ Visualização do endereço diretamente no **mapa**
* 📍 Detecção da **localização atual do usuário**
* 🧭 Botão para **iniciar navegação no Google Maps**
* 📱 Interface otimizada para **celular**
* ⚡ Estrutura preparada como **PWA (Progressive Web App)**

---

## 🧰 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Leaflet
* OpenStreetMap
* BrasilAPI
* Nominatim (OpenStreetMap Geocoding)

---

## 🌎 APIs utilizadas

* https://brasilapi.com.br
* https://nominatim.openstreetmap.org
* https://www.openstreetmap.org

---

## 📱 Como usar

1. Digite um **CEP válido** no campo de busca
2. Clique no botão de pesquisa
3. O endereço será exibido e marcado no mapa
4. Use o botão **Iniciar GPS** para abrir a rota no Google Maps

Também é possível usar o botão de **localização atual** para descobrir seu CEP aproximado.
