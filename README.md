# 📍 Busca CEP com Mapa

Aplicação web simples que permite **buscar um CEP e visualizar sua localização no mapa**, além de iniciar a navegação diretamente no GPS do celular.

O projeto utiliza **Leaflet com OpenStreetMap**, consulta dados de CEP pela **BrasilAPI** e realiza geocodificação usando **Nominatim**.

---

## 🌐 Acessar o site Versão DEMO

Você pode usar a aplicação diretamente pelo link abaixo:

🔎 **Buscar CEP online:**
(https://buscacepbrasil.netlify.app/)

---

## 🚀 Funcionalidades

* 🔍 Buscar endereço utilizando **CEP*
* 📍 Mostrar **localização atual do usuário**
* 🧭 Botão para **iniciar navegação no GPS (Google Maps)**
* 📱 Interface otimizada para **celular**
* ⚡ Estrutura compatível com **PWA (Removida)**

## 📱 Como usar

1. Digite um **CEP válido** no campo de busca
2. Clique no botão de pesquisa
3. O endereço será exibido no mapa
4. Use o botão **Iniciar GPS** para abrir a navegação no mapa

Também é possível clicar no botão de **localização atual** para descobrir seu CEP aproximado.
